# AGENTS.md

## Project Overview

This project is the **SCRAPZ TATZ** website for a tattoo business.

The site should feel:
- Dark
- Modern
- Artistic
- Custom
- Trustworthy

The visual identity should be driven by the **SCRAPZ TATZ logo** and its edgy electric blue/purple energy. Avoid anything that looks like a generic website template.

## Core Build Rules

- Use plain `HTML`, `CSS`, and `JavaScript` unless the user explicitly asks for a different stack.
- Keep code clean, readable, and easy to edit.
- Favor simple structure over unnecessary abstraction.
- Keep files organized and naming predictable.
- Avoid heavy dependencies unless clearly requested.

## Brand Direction

### Visual Tone

- Edgy but professional
- Artistic without becoming chaotic
- Bold and custom rather than corporate or generic
- High-contrast and immersive

### Color Direction

Primary environment:
- Near-black backgrounds
- Charcoal surfaces

Accent palette:
- Electric blue
- Purple
- Icy blue

Guidance:
- Use bright accents intentionally so they feel premium and energetic.
- Preserve strong contrast for readability.
- Do not let the palette drift into soft pastel, beige, or generic SaaS styling.

### Logo Relationship

- The SCRAPZ TATZ logo is the main visual anchor.
- Pull visual cues from the logo’s electric blue/purple energy.
- Supporting UI should complement the logo, not overpower it.
- Any glow, gradient, or highlight treatment should feel intentional and consistent with the logo.

## Design Guidance

- Prefer custom composition, bold spacing, and intentional typography.
- Use layered dark backgrounds, subtle gradients, textures, or glow accents where appropriate.
- Keep layouts polished and modern without looking overdesigned.
- Avoid stock “business template” patterns unless adapted to feel original.
- Ensure the site still feels trustworthy and easy to navigate.

## UX Guidance

- Make important information easy to find.
- Prioritize clarity for services, portfolio/work, artist identity, contact information, and booking flow.
- Preserve strong readability on dark backgrounds.
- Ensure good mobile responsiveness from the start.
- Use motion sparingly and purposefully.

## Implementation Preferences

- Prefer semantic HTML.
- Keep CSS modular and easy to scan.
- Use CSS variables for brand colors, spacing, and reusable visual tokens.
- Keep JavaScript lightweight and focused on interaction only where needed.
- Avoid overengineering for small features.

## Default Styling Constraints

- Backgrounds should generally stay in the near-black / charcoal range.
- Accent colors should come from electric blue, purple, and icy blue families.
- Buttons, links, hover states, borders, and highlights should reinforce the brand palette.
- Shadows, glows, and gradients should be subtle to medium strength, not noisy.
- Typography should feel deliberate and modern, not default-browser plain.

## What To Avoid

- Generic template styling
- Bright white page backgrounds
- Random color palettes unrelated to the brand
- Overly playful or cartoonish UI
- Overly corporate or sterile layouts
- Framework-heavy solutions when plain HTML/CSS/JS will do
- Messy, hard-to-edit code

## When Making Changes

- Keep the brand identity consistent across all sections.
- Match new UI to the existing dark, electric, artistic aesthetic.
- Preserve readability and usability while pushing for a distinctive look.
- Prefer maintainable solutions that the user can easily tweak later.

## Deliverable Standard

Any new page, section, or component should feel like it belongs to a premium custom tattoo brand site for SCRAPZ TATZ, with dark atmospheric styling and electric blue/purple energy at the center of the experience.
